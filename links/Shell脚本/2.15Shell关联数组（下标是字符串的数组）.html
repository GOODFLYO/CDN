Shell关联数组（下标是字符串的数组）

< 上一页Shell删除数组元素 Shell内建命令下一页 >

C语言中文网推出辅导班啦，包括「C语言辅导班、C++辅导班、算法/数据结构辅导班」，全部都是一对一教学：一对一辅导 + 一对一答疑 + 布置作业 + 项目实践 + 永久学习。QQ在线，随时响应！

现在最新的 Bash Shell 已经支持关联数组了。关联数组使用字符串作为下标，而不是整数，这样可以做到见名知意。

关联数组也称为“键值对（key-value）”数组，键（key）也即字符串形式的数组下标，值（value）也即元素值。

例如，我们可以创建一个叫做 color 的关联数组，并用颜色名字作为下标。 
复制纯文本复制
  1. declare -A color
  2. color["red"]="#ff0000"
  3. color["green"]="#00ff00"
  4. color["blue"]="#0000ff"
declare -A color
color["red"]="#ff0000"
color["green"]="#00ff00"
color["blue"]="#0000ff"
也可以在定义的同时赋值： 
复制纯文本复制
  1. declare -A color=(["red"]="#ff0000", ["green"]="#00ff00", ["blue"]="#0000ff")
declare -A color=(["red"]="#ff0000", ["green"]="#00ff00", ["blue"]="#0000ff")
不同于普通数组，关联数组必须使用带有-A选项的 declare 命令创建。关于 declare 命令的详细用法请访问：Shell declare和typeset命令：设置变量属性

访问关联数组元素

访问关联数组元素的方式几乎与普通数组相同，具体形式为：

array_name["index"]

例如： 
复制纯文本复制
  1. color["white"]="#ffffff"
  2. color["black"]="#000000"
color["white"]="#ffffff"
color["black"]="#000000"

加上$()即可获取数组元素的值：

$(array_name["index"])

例如： 
复制纯文本复制
  1. echo $(color["white"])
  2. white=$(color["black"])
echo $(color["white"])
white=$(color["black"])

获取所有元素的下标和值

使用下面的形式可以获得关联数组的所有元素值：

${array_name[@]}
${array_name[*]}

使用下面的形式可以获取关联数组的所有下标值：

${!array_name[@]}
${!array_name[*]}

获取关联数组长度

使用下面的形式可以获得关联数组的长度：

${#array_name[*]}
${#array_name[@]}


关联数组实例演示： 
复制纯文本复制
  1. #!/bin/bash
  2.  
  3. declare -A color
  4. color["red"]="#ff0000"
  5. color["green"]="#00ff00"
  6. color["blue"]="#0000ff"
  7. color["white"]="#ffffff"
  8. color["black"]="#000000"
  9.  
  10. #获取所有元素值
  11. for value in ${color[*]}
  12. do
  13.     echo $value
  14. done
  15. echo "****************"
  16.  
  17. #获取所有元素下标（键）
  18. for key in ${!color[*]}
  19. do
  20.     echo $key
  21. done
  22. echo "****************"
  23.  
  24. #列出所有键值对
  25. for key in ${!color[@]}
  26. do
  27.     echo "${key} -> ${color[$key]}"
  28. done
#!/bin/bash

declare -A color
color["red"]="#ff0000"
color["green"]="#00ff00"
color["blue"]="#0000ff"
color["white"]="#ffffff"
color["black"]="#000000"

#获取所有元素值
for value in ${color[*]}
do
    echo $value
done
echo "****************"

#获取所有元素下标（键）
for key in ${!color[*]}
do
    echo $key
done
echo "****************"

#列出所有键值对
for key in ${!color[@]}
do
    echo "${key} -> ${color[$key]}"
done
